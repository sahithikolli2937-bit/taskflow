# \# TaskFlow

# 

# TaskFlow is a simple web application that helps users organize daily tasks.

# 

# \## Features

# 

# \- Add a task

# \- Select task priority

# \- Display the latest task

# \- View today's task summary

# \- Organize tasks into categories

#

# \- CSS

# \- JavaScript
